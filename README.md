If the program fails to start:

* Restart your computer to close background tasks.
* Run the installer as an administrator by right-clicking the file and choosing Run as administrator.
* Check your antivirus software. Some security tools block installers temporarily. Disable your security software for one minute, run the installer, and then enable your security software again.

If you cannot find a program in the list:

* Refresh the list by clicking the view menu inside the application.
* Use the search bar at the top right to filter by name.
* Ensure the program is not a portable application, as portable software does not always show up in standard installation lists.
